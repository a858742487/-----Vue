## Build Setup
  
``` bash
# 克隆项目
git clone https://github.com/a858742487/Liuguofu-Zhihu-Vue.git

# 安装依赖     
npm install

# 建议不要用cnpm安装 会有各种诡异的bug 可以通过如下操作解决 npm 下载速度慢的问题
npm install --registry=https://registry.npm.taobao.org

# 运行 localhost:8080    
npm run dev

# 打包
npm run build

```
